# apipost-idea-plugin
## apipost idea2026.1版本插件
初衷：官方插件好久没更新了，在idea2025版本上勉强还可以用，但是升级idea2026版本后彻底不可使用，询问了官方最近的预期更新计划要到4月底才可能会发布更新包支持idea2026版本，本人日常工作中重度使用等不及了所以基于官方插件进行了调整更新，修复了会出现的错误问题能够支持idea2026版本<br>
具体更新如下：<br>
1、取消官方插件的更新检测，那个更新检测有问题明明没有更新结果每次启动idea都提示有新的插件更新弹窗<br>
2、修复在2025和2026版本中经常出现的异常报错不兼容问题<br>
3、调整插件版本用于与官方发布版本进行区分<br>
4、最低支持为2024.1版本，最高暂未限制我本人测试版本为2026.1<br>

### 特殊说明，此插件为本人基于官方插件调整后的用于本人日常工作使用。此插件不进行任何盈利行为，如官方更新插件后请各位及时更新官方插件
### 本人主要用于进行接口上传操作，平日调试接口不使用apipost插件，各位按需使用！
